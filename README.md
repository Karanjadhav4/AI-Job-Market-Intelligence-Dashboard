# AI Job Market Intelligence Dashboard

## 📌 Project Overview

The **AI Job Market Intelligence Dashboard** is a real-time data science project that analyzes the demand for data-related roles by scraping job listings from LinkedIn.
The system collects job data, processes it, and visualizes hiring trends through an interactive Streamlit dashboard.

This project helps job seekers, analysts, and researchers understand the **current job market demand for Data Science roles** such as Data Scientist, Machine Learning Engineer, AI Engineer, Data Analyst, and Data Engineer.

---

## 🎯 Objectives

* Scrape real-time job postings from LinkedIn
* Clean and structure job listing data
* Analyze hiring trends across companies and locations
* Extract useful insights about the job market
* Provide an interactive dashboard for visualization

---

## 🚀 Features

* Real-time LinkedIn job scraping
* Automated data cleaning and preprocessing
* Interactive data visualization using Streamlit
* Company-wise and location-wise hiring analysis
* Downloadable dataset for further analysis
* Simple user interface for exploring job trends

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Karanjadhav4/AI-Job-Market-Intelligence-Dashboard
```

Navigate to the project folder:

```
cd AI Job Market Intelligence Dashboard
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Start the Streamlit application:

```
streamlit run app.py
```

Then open your browser and go to:

```
http://localhost:8501
```
